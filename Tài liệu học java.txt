2-Kiểu số thực
3-Kiểu dữ liệu tham chiếu
array : Cấu trúc mảng của các dữ liệu cùng kiểu 
class : Dữ liệu kiểu lớp đối tượng do người dùng định nghĩa, bao gồm các thuộc tính và phương thức 
interface : kiểu dữ liệu lớp giao tiếp do người dùng định nghĩa, khai báo các phương thức của giao tiếp



*Các quy tắc chuyển kiểu dữ liệu 
Có 2 quy tắc chuyển đổi :
1-Mở rộng kiểu : Giá trị của kiểu hẹp hơn được chuyển sang giá trị của kiểu rộng hơn, có số phần tử nhiều hơn
VD: 
int a = 5; double b = 9.4;
b = a; //Mở rộng kiểu tự động
a = int(b); //Phần thập phân sẽ bị bỏ

2-Thu hẹp hay ép kiểu : Chuyển đổi kiểu theo chiều ngược lại của mở rộng kiểu, từ kiểu rộng về kiểu hẹp hơn được gọi là thu hẹp kiểu
VD:

long n=120;
int k = int(N); //ép kiểu long về int 
float f = 2.5f; 	//ép 2.5 mặc định kiểu double về float
cb = (Canbo)ns; //ép ns là đối tượng lớp Nhansu về lớp Canbo
